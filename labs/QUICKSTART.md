# Quickstart

## 1) Start Environment

```bash
docker compose up -d
docker compose ps
```

## 2) Run First Lab

```bash
cd /home/mark/projects/ES/Advanced-Kafka-Streaming
less labs/01-Modern-EDA/lab-01-kafka-topology.md
```

## 3) Follow Suggested Lab Order

```text
Lab 1 -> Lab 2 -> Lab 3 -> Lab 4 -> Lab 5 -> Lab 6 -> Lab 7 -> Lab 8 -> Lab 9 -> Lab 10
```

## 4) Clean Up

```bash
docker compose down
```
